## Installation

You can install the package via composer:

```bash
composer require furkifor/laravel_logging_url
```

## Usage

```php
$laravel_logging_url = new Furkifor\LaravelLoggingUrl();
echo $laravel_logging_url->echoPhrase('Hello, Furkifor!');
```
